# Kote pou mete APK Loré School la

Mete fichye `.apk` ou jwenn nan EAS Build la isit la, epi renome l
`lore-school.apk` (menm non ki nan `lib/school-app-config.ts`):

```
public/downloads/lore-school.apk
```

Depi fichye a la, lyen "Télécharger pour Android" nan seksyon
telechajman sou paj akèy la ap mache otomatikman — pa gen okenn lòt
konfigirasyon pou fè.

Chak fwa ou gen yon nouvo vèsyon:
1. Ranplase fichye `lore-school.apk` la ak nouvo a (menm non an)
2. Louvri `lib/school-app-config.ts`, mete ajou `version` (ak
   `approxSizeMb` si l chanje anpil)
3. Deplwaye sit la ankò
